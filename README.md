# resume

## Contains
- [x] Yarn
- [x] PostCSS
 - [x] autoprefixer
 - [x] postcss-custom-properties
 - [x] postcss-import
 - [x] postcss-nested
 - [x] stylelint
 - [x] stylefmt

## Development

```sh
yarn install   # Init

yarn css       # Build css
yarn w-css     # Watch css
yarn lint-css  # Lint css

yarn deploy    # Deploy to gh-pages
```
